How AeroVerse Works

What is AeroVerse?

AeroVerse is a where Blockchain meet Mixed Reality. In Aeroverse users can collect Aeroverse (AERO) tokens, NFTs & in-game items or trade Lands & Billboards in the mixed reality world.

How does Aeroverse works ?

Aeroverse is a AR(Augmented Reality) based game, where users can find crypto tokens & NFTS in real world around. Users can go to specific location based GeoPools & battle with others to win chance to earn exclusive legendary NFTs.
Users can bid/buy Lands, Billboards or premium spots in Metaverse of Aeroverse
Also users can avail many more features such as stake AERO Tokens, Farm tokens & NFTs to win Cosmic Balls & other in game rewards.

What is Aeroverse GeoPool?

AeroVerse GeoPools is a unique venue for any ERC20/BEP20 compatible tokens, in-game items & NFTs.
Each GeoPool will have partner agencies who can sell/rent out spots on the Geopool to earn more AERO Tokens. Each Geopool emits unique CosmicBalls when liquidity is deposited. The portion of GeoPool's liquidity provided determines, number of CosmicBalls user receives.
Each Geopool has its fixed latitude & longitude & a contract address
Geopool gets CelestialDraw at-least twice every week for users on FCFS basis

How does Celestial Draw works ?

CelestialDraw contains 10 items including few rare & normal NFTs, several Crypto Tokens & in game rewards
Each draw has an algorithm which defines probability of user getting each item
First pull is fixed at 10 CosmicBalls and with each pulls the cost of next pull increases by a predefined algorithm
User can get rare/special NFT at any pull, item unlocking on each pull is purely random & based on probability calculations of item getting picked
Each Celestial Draw has fixed number of rewards for NFT/Tokens/CosmicBalls.
If any item is fully claimed then it is removed from draw while user can still play Celestial Draw for other rewards

How does Exclusive Drop Battle works ?

Whenever an exclusive NFT is dropped on any Geopool, user need to whitelist by depositing min required CosmicBalls on FCFS basis
on time of exclusive drop, whitelisted users will be given upto 3 tasks to perform in game (i.e. visit X geopools, Farm n more tokens etc).
N number of users who performs the tasks first (based on block timestamp) will be announced winner to claim Exclusive drop NFTs.
Remaining whitelisted users who did not win, Deposit will be returned to their wallets.

What is Cosmic Ball ?

Each cosmic ball contains stardust.
Stardust energy is required during battles for Exclusive NFT drop.

What is AeroVerse Realtor ?

User can buy CosmicBalls, AERO Token or Premium Land spot, Billboards in Aeroverse Realtors
Users can sell/auction their NFTs in AeroVerse
Users can swap tokens on Swaps
Users can buy or search for NFTs in Realtors
